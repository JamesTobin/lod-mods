#Legendary_Mese

Adds armor and tools made of Legendary Mese

It is licensed under an MIT license

Dependencies: Default, 3d_armor, more_mese

This is currently version 1.0 of this mod

To install, go to https://github.com/gitgithu/legendary_mese

This mod was created for the Legends of Dragons server, and it will NOT work if you do not have the more_mese mod (as shown in the dependencies)
